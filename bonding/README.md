# Bonding Module

This directory contains all bonding-specific modules for multi-channel OpenVPN bonding.

## Structure

- `include/` - Header files for bonding components
- `src/` - Implementation files
- `service/` - Windows service component
- `ui/` - GUI integration
- `utils/` - Utility functions
- `tests/` - Unit tests

## Building

The bonding module is automatically included when building the main OpenVPN GUI project.
